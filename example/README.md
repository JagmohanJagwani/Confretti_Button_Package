## ✨ Features

- 🎈 Bubble explosion animation
- 🚀 No external confetti package
- ✨ Lightweight and reusable
- 🎨 Customizable button color
- 📱 Supports Android, iOS, Web, Windows, macOS & Linux

# 🚀 Installation

Add this to your `pubspec.yaml`:

```yaml
dependencies:
  confetti_button: ^1.0.1
```

Run:

```bash
flutter pub get
```

---

# 📦 Import

```dart
import 'package:confetti_button/confetti_button.dart';
```
